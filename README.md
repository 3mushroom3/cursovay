# kasabov_dgtu

Flutter + Firebase приложение для сбора предложений.

## Запуск

```bash
flutter pub get
flutter run
```

## Заметки

- Конфигурация Firebase хранится в `android/app/google-services.json` и `lib/firebase_options.dart`.
- Для iOS используется `ios/Runner/GoogleService-Info.plist`.
